# MMD

`MMD` 是一个基于 `Tauri + React` 打造的桌面 Markdown 编辑器，面向写作、提示词整理、代码片段记录与本地文本工作流。

它强调一件事：让 Markdown 编辑保持轻、快、稳定，同时具备足够现代的桌面体验。打开即用，默认双栏编辑与预览，适合长期停留在本地文件体系中的内容创作场景。

## 产品特点

- 桌面原生体验，启动轻量，适合日常高频使用
- 默认双栏布局，编辑与预览同时进行
- 面向本地文件工作流，支持直接打开和保存 Markdown / 文本文件
- 保持 UTF-8 读写，适合中英文混合内容与代码片段管理
- 提供常用 Markdown 格式化能力，减少重复手写
- 支持拖拽打开文件与双击关联文件启动
- 提供明暗主题切换，更适合长时间阅读与编辑

## 适用场景

- Markdown 笔记与文档写作
- 命令行片段、脚本片段、开发备忘记录
- 本地知识库与轻量文稿编辑

## 技术栈

- `Tauri`
- `React`
- `TypeScript`
- `Vite`
